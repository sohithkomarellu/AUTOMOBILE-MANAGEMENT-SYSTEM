# Automobile Management System

A simple web application built using **Java, JSP, JDBC, and MySQL** to manage vehicle services and user information. The system allows admins, service centers, and users to access features based on their roles.

## Features
- **Admin:** Login, add/view services, view all users
- **Service Center:** Login, add car details, manage service information
- **User:** Register/Login, view profile, search car details, view basic analysis

## Technologies Used
- Java (JDK 8+)
- JSP & Servlets
- JDBC
- MySQL Database
- Apache Tomcat Server
- HTML, CSS, JavaScript
